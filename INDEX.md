## ðŸ†˜ Emergency Help

### Server Won't Start
1. Run `QUICK_CHECK.bat`
2. Check `QUICK_REFERENCE.md` > Quick Fixes
3. Read `FIX_SUMMARY.md` > Server Issues

### Login Not Working
1. Check server is running (port 8081)
2. Verify database has test user
3. Check browser console (F12)
4. Review `DEBUG_CHECKLIST.md`

### Database Errors
1. Verify MySQL is running
2. Run `backend/FINAL_FIX.sql`
3. Check credentials in `DatabaseConnection.java`
4. Read `SETUP_GUIDE.md` > Database Setup

### Build Fails
1. Verify Java and Maven installed
2. Clean build: `mvn clean install`
3. Check `COMPLETE_FIX_GUIDE.md`
4. Review error messages
